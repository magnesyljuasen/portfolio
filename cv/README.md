# CV

Den publiserte CV-en ligger i `public/cv/magne-syljuasen-cv.pdf`.

Kildeteksten og utformingen ligger i `generate_cv.py`. Etter en tekstendring kan PDF-en bygges på nytt fra rotmappen med:

```powershell
python cv/generate_cv.py
```

## Historien CV-en bygger på

Magne er en tverrfaglig sivilingeniør som har beveget seg fra geofag og praktisk feltarbeid til energi, dataanalyse og digital produktutvikling. Styrken ligger i å lære nye fagområder, se helheten, strukturere komplekse problemer og gjøre beregninger om til løsninger som faktisk blir brukt.

De viktigste bevisene er:

- AV Energiplanlegging er brukt i mer enn 20 kundeprosjekter.
- Bergvarmekalkulatoren ble solgt til Norsk Varmepumpeforening.
- GeoTermos-kalkulatoren ble solgt til Hafslund.
- Analysemodellene er brukt i beslutningsgrunnlag for blant annet Oslo kommune, Enova, UiT og Kristiansand kommune.

CV-en er bevisst uten bilde, fødselsdato, full gateadresse og videregående skole. Det gir mer plass til resultater, relevante søkeord og dokumentert verdi.
